# 前端页面架构

## 项目概述

本项目是一个基于HTML、CSS和JavaScript的前端应用，主要用于演示MCS8 JavaScript Client SDK的功能。项目包含两个主要页面：`index.html`（开发指南）和`demo.html`（快速上手示例）。

## 文件结构

```
camera-surveillance/
├── index.html          # 开发指南主页面
├── demo.html           # 快速上手示例页面
├── style.css           # 全局样式文件
├── package.json        # 项目依赖配置文件
├── doc/                # 文档目录
│   ├── MCS8_WebAPI_V2.pdf
│   └── 数据库业务接口.pdf
├── img/                # 图片资源目录
│   ├── img_1.jpg
│   └── img_2.jpg
├── js/                 # JavaScript库和框架目录
│   ├── crypto-js.js
│   ├── jquery.min.js
│   ├── template-web.js
│   ├── bootstrap-3.4.1-dist/
│   └── zTree/
├── jssdk/              # MCS8 JavaScript SDK目录
│   ├── mcs8Client.js
│   └── sdkDemo.js
└── node_modules/       # Node.js依赖目录
```

## 核心页面

### 1. index.html - 开发指南

这是主要的开发指南页面，提供了MCS8 JavaScript Client SDK的完整使用说明。

#### 页面结构
- **导航栏**: 包含页面标题和下载链接
- **左侧菜单**: 按功能分类的SDK使用指南目录
- **中间内容区**: 显示选中功能的详细说明和示例代码
- **右侧日志区**: 显示操作日志

#### 主要功能模块
1. 快速上手
2. WEBSDK-连接网关
3. WEBSDK-实时视频/对讲
4. WEBSDK-GPS/设备参数
5. WEBSDK-消息上报
6. WEBSDK-指令下发

### 2. demo.html - 快速上手示例

这是一个简化版的示例页面，引导用户快速上手SDK使用。

#### 页面结构
- **步骤导航**: 连接网关和获取设备列表两个主要步骤
- **操作面板**: 显示设备信息和功能按钮
- **日志区域**: 显示操作日志

## 核心技术栈

1. **HTML5**: 页面结构和语义化标签
2. **CSS3**: 样式设计，使用Bootstrap框架
3. **JavaScript**: 业务逻辑实现
4. **jQuery**: DOM操作和事件处理
4. **Bootstrap 3.4.1**: UI框架
5. **zTree**: 树形结构展示组件
6. **MCS8 JavaScript Client SDK**: 核心功能SDK

## 样式和资源

### style.css
全局样式文件，主要定义了：
- 菜单和内容区域的布局
- 代码块样式
- 视频元素样式

### 图片资源
- `img_1.jpg`和`img_2.jpg`: 证书错误提示页面截图

## JavaScript组件

### 第三方库
1. `jquery.min.js`: jQuery库
2. `template-web.js`: JavaScript模板引擎
3. `crypto-js.js`: 加密算法库
4. `bootstrap-3.4.1-dist/`: Bootstrap框架文件
5. `zTree/`: 树形结构展示组件

### 自定义SDK
1. `mcs8Client.js`: MCS8客户端SDK核心文件
2. `sdkDemo.js`: SDK使用示例和功能实现

## 功能特点

1. **模块化设计**: 每个功能模块都有独立的HTML模板和JavaScript函数
2. **响应式布局**: 使用Bootstrap实现响应式设计
3. **实时通信**: 通过MCS8 SDK实现与设备的实时通信
4. **多媒体支持**: 支持视频播放、音频监听和对讲功能
5. **设备管理**: 支持设备列表展示、状态监控和参数获取
6. **消息处理**: 支持GPS消息、设备状态消息和报警消息的处理
7. **指令下发**: 支持向设备下发录像、工单和文本消息等指令
